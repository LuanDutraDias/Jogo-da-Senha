# Jogo da Senha – C++
- Projeto desenvolvido na disciplina de Algoritmos (1º período – Ciência da Computação).
- Implementação do clássico jogo da senha em C++, utilizando apenas variáveis simples (sem vetores ou matrizes).

## Descrição
- O programa:
- Gera uma senha secreta de 4 dígitos (valores de 1 a 6)
- Permite múltiplas tentativas
- Fornece feedback após cada jogada:
- O → dígito correto na posição correta
- X → dígito correto na posição errada
- _ → dígito inexistente

## Conceitos Aplicados e Aprendizados
- Estruturas condicionais (if/else)
- Estruturas de repetição (for, do-while)
- Manipulação de dígitos com divisão inteira (/) e módulo (%)
- Modularização com funções
- Validação de entrada do usuário
- Geração de números aleatórios
- Pensamento algorítmico
- Tratamento de entradas inválidas
